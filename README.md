# H20IQ

### Version
1.0.0

### Installation and Setup

* Install Python 3.x.x
* In a terminal window:

    ```sh
    $ pip install django
    $ cd root/for/H20IQ
    $ git clone (repo address)
    $ cd H20IQ
    $ H20IQ> python manage.py migrate
    $ H20IQ> python manage.py createsuperuser
    ```
* Follow the prompts
